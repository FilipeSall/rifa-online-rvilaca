Como usar no Firebase:

1. Abra Firebase Console > Authentication > Templates > Email address verification.
2. Defina o assunto acima.
3. Cole o HTML de `docs/auth/firebase-email-verification-template.html` no campo de mensagem.
4. Salve e envie um email de teste pelo proprio console.

Placeholders usados no template:

- `%APP_NAME%`
- `%EMAIL%`
- `%LINK%`
